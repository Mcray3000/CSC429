\section{3. Security and Privacy Applications}

\textbf{Application 1: Federated learning in IoT devices.}

Federated Learning (FL) allows IoT devices to collaboratively train machine learning models using local data, such as vehicle sensor data or user network traffic, without sending raw data to a central server. ML is very useful here because it enables distributed networks to learn from vast amounts of real-world data while strictly preserving user privacy, as only the computed model updates are shared rather than the sensitive data itself.

\textbf{Application 2: Privacy-preserving intrusion detection in IoT.} 

For industrial IoT environments, it is important to have robust Intrusion Detection Systems. However, oftentimes industrial network traffic is sensitive, and transferring it to a central server risks exposing proprietary operational data.  Machine Learning is useful here because models, such as a 1D CNN, can be deployed to monitor and classify malicious traffic (like a DoS) locally.
